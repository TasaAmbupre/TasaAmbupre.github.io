三角函数核心概念

三角函数是研究三角形和周期现象的重要工具，主要包含：

  • 正弦函数 sinθ = 对边/斜边
  • 余弦函数 cosθ = 邻边/斜边
  • 正切函数 tanθ = 对边/邻边

单位圆关系：

当r=1时：
x = cosθ
y = sinθ
tanθ = y/x

常用恒等式

sin²θ + cos²θ = 1
1 + tan²θ = sec²θ
sin(A±B) = sinAcosB ± cosAsinB
cos(A±B) = cosAcosB ∓ sinAsinB

定理与法则

正弦定理：
a/sinA = b/sinB = c/sinC = 2R
余弦定理：
c² = a² + b² - 2abcosC
诱导公式：
sin(π/2-θ) = cosθ
cos(π/2-θ) = sinθ
